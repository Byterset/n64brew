Renders a part of a Sausage64 model. Calculates the model-\/view transformation matrix for the part based on position, rotation, and scale in the frame data. Handles interpolation between current and next frame data if enabled. 

For billboard parts, calculates a rotation matrix based on the current view matrix. For other parts, calculates a rotation matrix from the quaternion in the frame data.

Multiplies the matrices together, converts to a N64 matrix, and sets it as the model-\/view matrix before drawing the display list for the part mesh.
