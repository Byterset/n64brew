\doxysubsection{Detailed Description}
S64 Model Helper. Holds the model data, the current animation, the current keyframe, the current animation tick, the current matrix, the pre draw and post draw functions and the model data. 

\begin{DoxyNote}{Note}
This is used to actually create and hold a Sausage64 Model and it\textquotesingle{}s data. Initialize it using sausage64\+\_\+initmodel 
\end{DoxyNote}
